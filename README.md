# Game YEP - Landing Page

Landing page cho game "Vào Lệnh Không Run"

## Cài đặt

1. Cài đặt dependencies:
```bash
npm install
```

## Chạy dự án

Chạy development server:
```bash
npm run dev
```

Mở trình duyệt tại `http://localhost:5173`

## Build

Để build production:
```bash
npm run build
```

File build sẽ nằm trong thư mục `dist`

## Preview production build

```bash
npm run preview
```
